# 🚗 Car Price Prediction App

A Machine Learning web application that predicts the resale price of a car based on mileage, car age, and brand.  
The application is built using **Python, Machine Learning, and Streamlit**.

---

## 📌 Project Overview

This project uses a trained Machine Learning model to estimate the resale value of a car.  
Users can input car details such as brand, mileage, and age, and the system predicts the estimated price.

The goal of this project is to demonstrate the use of **Machine Learning models integrated with a web interface**.

---

## ⚙️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

---

## 📊 Features

- Car brand selection
- Mileage input
- Car age input
- Machine Learning price prediction
- Interactive web interface
- Clean dashboard design
